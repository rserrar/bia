# Local Frontend

Conté l'aplicació local de monitoratge i visualització de runs i resultats.

## Execució

- Mode continu: `python local-frontend/src/run_monitor.py`
- Mode consulta única: definir `V2_MONITOR_ONCE=true`

Variables:

- `V2_API_BASE_URL`
- `V2_RUN_ID`
- `V2_POLL_SECONDS`
- `V2_MONITOR_ONCE`
